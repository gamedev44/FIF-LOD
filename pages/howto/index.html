How to Use

Training with Scenarios

  1. Click "Begin Training" from the home page
  2. Select a nursing category (Anatomy, Pharmacology, Med-Surg, etc.) or load a custom JSON file
  3. Read the question and select your answer
  4. Tap the card to flip and see the detailed explanation
  5. Tap again to continue to the next question
  6. Use the ‚Üê button to review previous questions
  7. Track your score to monitor your knowledge retention

Loading Scenario Files

You can load scenario sets by:

  1. Select from categories: Choose a category from the dropdown menu
  2. Drag and drop: Drag a JSON file onto the training page to load it

Creating Your Own Scenarios

Edit the category HTML files in pages/ and add to the flashcardDecks object. Perfect for creating custom scenario sets for specific exams or topics:

File-level Fields:

  • title - Category name (e.g., "Cardiac Nursing") (required)
  • test - Test number for this quiz set (required)
  • category - Category name matching card categories (required)
  • cards - Array of scenario objects (required)

Card-level Fields:

  • id - Question ID (q1, q2, etc.) (required)
  • category - Category name (required)
  • test - Test number (required)
  • scenario - Emergency scenario text (optional but recommended)
  • timer - Time limit in seconds (optional)
  • q - Question text (required)
  • choices - Array of answer choices (required)
  • correct - Index of correct answer (0-based, required)
  • a - Answer explanation with nursing rationale (required)
  • successMessage - Message shown on correct answer (optional)
  • failureMessage - Message shown on wrong answer (optional)
  • causeOfDeath - Cause of death for death certificate (optional)
  • timeoutMessage - Message shown when time runs out (optional)
  • image - Optional image path (anatomy diagrams, medication images, etc.)
  • answerImage - Optional image path for answer side
