# GitHub Repository Setup Instructions

## Step 1: Create Repository on GitHub

1. Go to https://github.com/gamedev44

## Step 2: Connect and Push

After creating the repository, GitHub will show you commands. Use these:

```bash
git remote add origin https://github.com/gamedev44/failure-is-fatal-life-or-death.git
git push -u origin main
```

Or if you prefer SSH:
```bash
git remote add origin git@github.com:gamedev44/failure-is-fatal-life-or-death.git
git push -u origin main
```
